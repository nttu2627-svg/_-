任務——從區域選項中挑出唯一且最貼切的地點，以便完成當前行動。

小鎮可用場景速覽：
- Apartment_F1：公寓一樓住宅，多數角色住所。
- Apartment_F2：公寓二樓住宅，亦可視為角色住所延伸。
- School：學校室內空間，進行教學、備課、查資料等工作與學習。
- Rest：餐廳／咖啡廳，可用於用餐、社交或短暫休息。
- Gym：健身房室內場景，供鍛鍊訓練。
- Super：超級市場室內空間，採購補給與日用品。
- Subway：地鐵站室內場景，交通樞紐與移動節點。
- Exterior：泛指街道、公園、廣場等露天區域。

主要室外建築與對應室內場景：
- 公寓 → Apartment_F1 / Apartment_F2。
- 學校 → School。
- 餐廳 → Rest。
- 超市 → Super。
- 健身房 → Gym。
- 地鐵 → Subway。
- Park → Exterior（亦可包含其他露天地點）。
- 醫院 → 正在維修，暫不可用；除非任務強制且列表僅餘此選項，否則不要選擇。

與 schedules.json target 對照的推薦策略：
- "Apartment_F1" 或家庭相關行動 → 優先選擇 Apartment_F1，若不可選再考慮 Apartment_F2 或其他住所場景。
- "School" → 前往 School。
- "Rest" → 選擇 Rest（餐廳／咖啡廳）。
- "Gym" → 選擇 Gym。
- "Super" → 選擇 Super。
- "Exterior" → 選擇 Exterior，例如公園散步、戶外活動。
- 若未來出現 "Subway" → 選擇 Subway。
- 若任務文本指向醫院但該場景不可用 → 回報無法執行並改選可行替代（如留在現地或選擇可支援任務的其他場所）。

決策規則：
1. 確認可選清單，輸出必須與清單中的場景名稱完全一致（大小寫與下劃線皆需匹配），且僅能輸出一個地點。
2. 若目前所在地已能完成任務，優先選擇留在原地。
3. 若任務對應 schedules.json 既定目標，依上述映射挑選；若無直接對應，根據場景功能選擇最能支援任務的地點。
4. 醫院場景暫時封閉；如任務可改在其他地點完成，應避開醫院。
5. 當所有候選都不合適時，選擇最接近任務需求或回到住所，但仍須從給定清單中挑選。
